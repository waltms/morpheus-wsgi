We are excited to include the time-honored Morpheus parser here as a fall-back or second opinion for our own static database, which is far from offering parses for all words in Greek literature. Morpheus was originally written by Gregory Crane for the Perseus project. This version comes from The Perseids Project Tools on GitHub. Web interface by Walter M. Shandruk. Note: please report incorrect parses and other malfunctions to us. Do bear with us as we try to understand this tool better!
